# Dungeon Survivor

A wave-based survival RPG game. Survive as long as you can against endless waves of dungeon creatures!

## How to Play

- **Move:** WASD or Arrow Keys (or drag on mobile)
- **Attack:** Auto-attack nearest enemy
- **Level Up:** Choose 1 of 3 skills
- **Goal:** Survive as many waves as possible!

## Game Features

- Wave-based enemy spawns
- 3 enemy types: Slime, Skeleton, Orc
- 6 upgradeable skills
- Boss battles every 5 waves
- Gold and XP collection
